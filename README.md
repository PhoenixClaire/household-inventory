# household-inventory

A fullstack web application that helps users track household items, monitor stock levels and generate a shopping list based on low inventory thresholds. 

This project was built to help keep track of household items and knowing when to restock essentials. 

Instead of manually checking supplies or forgetting what to buy, the system automatically tracks item quantities and generates a shopping list when items fall below a defined threshold. 

## Features
- add, update and delete household items
- set quantity thresholds for each item
- automatically generate a shopping list when items fall below threshold
- display required purchase quantity
- prevent duplicate items in inventory
- real-time updates after changes

## Tech Stack
### Frontend:
- React (Vite)
- TailwindCSS

### Backend:
- Node.js
- Express.js

### Database:
- MondoDB

### Deployment:
- Vercel (Frontend)
- Render (Backend)

## Setup
### 1. Clone the repository
git clone https://github.com/PhoenixClaire/household-inventory

### 2. Backend setup
cd backend
npm install
npm run dev

### 3. Frontend setup
cd frontend
npm install
npm run dev
